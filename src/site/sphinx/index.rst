**SQLSheet** is Excel JDBC Driver on top of Apache POI. It allows for reading and writing data from and into Excel sheets using only SQL statements:

- `SELECT FROM`
- `INSERT INTO`
- `DELETE FROM`
- `CREATE TABLE`
- `DROP TABLE`
